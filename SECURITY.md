# Security Policy

## Supported Versions

We actively support and fix vulnerabilities on the following versions of Opus Ludus:

| Version | Supported |
| ------- | --------- |
| v0.2.x  | Yes       |
| < v0.2  | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do not report it via public issues. Instead, report it privately to the maintainers by emailing:

**rodrigo.estrada@gmail.com**

Please include detailed information on the vulnerability, the potential impact, and instructions on how to reproduce the issue. We will acknowledge receipt of your report within 48 hours and work with you to resolve the issue as quickly as possible.
